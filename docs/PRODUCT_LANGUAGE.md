# Product Language

Use these terms consistently across all copy, UI labels, and code.

## Core Terminology

**Tab / Session Management**:
- "Open Tab" — Create a new drinking session (NOT "start session", "begin")
- "Close Tab" — End and archive a session (NOT "end session")
- "Join Tab" — Participate in existing tab (NOT "join session")
- "Active Tab" — Currently running, live tab
- "Tab History" — Archived/past tabs

**Drinks & Tracking**:
- "Add Drink" — Log a beverage (NOT "log drink", "track drink")
- "Pint Score" — Total eligible ounces ÷ 16
- "Bonus Drink" — Shot or cocktail (no oz count)
- "Beers", "Wine", "Seltzer" — Eligible drink types
- "Shot", "Cocktail" — Bonus drink types

**Social**:
- "Member" — Person in the tab
- "Join Code" — Alphanumeric code to join tab
- "Visibility" — Tab access level (Open, Invite Only, Private)

**Metrics**:
- "Pint Score" — oz / 16
- "Verified" — Drink has photo proof
- "Contribution" — User's total oz in a tab
- "Leaderboard" — Ranked list of users

## Tone & Style

- **Fun & Casual** — Embrace drinking culture
- **Friendly** — Use "you", second person
- **Concise** — Clear and direct
- **Scrapbook-like** — Celebratory, keepsake-like

## Example Usage

✅ **Correct**:
- "Open a tab to get started"
- "Mike added a beer · 16 oz"
- "You earned the Social Butterfly badge"

❌ **Incorrect**:
- "Start a session"
- "Mike logged a 16 oz beer"
- "New achievement unlocked"
