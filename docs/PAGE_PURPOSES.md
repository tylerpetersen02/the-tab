# Page Purposes

## Page Map & Navigation

5 main pages via bottom nav. Each has a distinct purpose.

### Feed (`/feed`)
**Purpose**: Social activity stream — see what friends are drinking, badges earned, tabs created.
**Content**: Drink posts, session posts, bonus posts, badge posts, recap posts
**Tone**: Social, celebratory, real-time

### Tabs (`/tabs`)
**Purpose**: Browse and manage tabs — see live, recent, yours, archived.
**Content**: Tab list, search, filter pills (All, Live, Recent, Mine, Archived)
**Actions**: Open tab, join code, create tab

### Add Drink / Current Tab Hub (`/add-drink`)
**Purpose**: Manage current tab & add drinks — the command center.
**Screens**:
  1. Gate (no active tab) — Open or Join buttons
  2. Create Tab — Form (title, photo, visibility, location, goal)
  3. Join Tab — Code entry
  4. Current Tab (Hub) — Hero, goal progress, contribution, members, breakdown, activity, actions
  5. Add Drink — Drink type, size, optional fields, post

**Key**: Creating/joining shows hub. Posting drink returns to hub. Leaving preserves history. Closing archives.

### Leaderboard (`/leaderboard`)
**Purpose**: Competitive rankings — who's on top by pint score & drink count.
**Content**: Time filter, stat card, top 3 podium, rank list, highlights
**Tone**: Competitive, achievement-focused

### Receipts (`/receipts`)
**Purpose**: Personal stats & achievements — review your history and earned badges.
**Screens**: Stats (beers, pints, oz, favorite) | Badges (earned & in-progress)
**Tone**: Reflective, celebratory

### Profile (`/profile`)
**Purpose**: User settings & preferences (future).

### Buddies (`/buddies`)
**Purpose**: Manage friends & contacts (future).

## Bottom Navigation

5 main tabs (always visible):
1. Feed → `/feed`
2. Tabs → `/tabs`
3. + → `/add-drink`
4. Leaderboard → `/leaderboard`
5. Receipts → `/receipts`

**Key Rules**:
- Each main tab has distinct purpose (no overlap)
- Sub-pages appear within parent tab flow
- Back button returns to main page
- Bottom nav always visible except side sheets
